Copy the HTML block below and paste it into your email client's signature settings (Gmail → Settings → Signature, Apple Mail → Preferences → Signatures, Outlook → Options → Mail → Signatures).

How to install in Gmail

  1. Open the preview file in a browser, select everything below the dashed line, and copy.
  2. Gmail → ⚙️ → See all settings → General → Signature.
  3. Click Create new, name it usdpr-primary, paste the signature into the editor.
  4. Set For new emails use and On reply/forward use to usdpr-primary. Save.

Optional — add when you set them up

  1. Phone: uncomment the <a href="tel:..."> line and paste your Google Voice / Twilio number.
  2. Calendly: swap the Start free trial button href for your Calendly URL.
  3. Everything else works as-is — email is the trusted default channel.
